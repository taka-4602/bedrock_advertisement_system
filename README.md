## 広告掲載の説明
Discordサーバー ベッドロックコミュニティー で掲載の申し込みができます  
https://discord.gg/w8Bxpb3BHk  
平日は1日あたり500円で広告を掲載できます  
週末(金曜 / 土曜 / 日曜)は1日あたり1000円で広告を掲載できます  
※料金は変更になる可能性があります  
また、同時期に広告を掲載したい人が複数人申し込んだ場合、オークション形式での料金設定になります  
## 注意点
- 料金は前払いとなります  
- 僕の独断で、掲載をご遠慮いただく可能性があります
- ベッドロックコミュニティーの広告が掲載されている期間は、広告掲載の申し込みが不可能な場合があります
